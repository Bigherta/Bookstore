# Bookstore 系统安装手册

---

## 1. 概述

本手册用于指导用户在本地环境中正确安装与部署 **Bookstore 书店管理系统**。系统采用 C++ 作为后端核心语言，基于 Qt 5.x 构建图形用户界面，适用于教学实验与课程设计场景，支持图书管理、用户管理及基础业务操作。

---

## 2. 运行环境要求

### 2.1 硬件环境

* 处理器：x86_64 架构
* 内存：≥ 4 GB（推荐 8 GB）
* 磁盘空间：≥ 1 GB 可用空间

### 2.2 软件环境

* 操作系统：

  * Windows 10 / 11（64 位）
  * Linux（Ubuntu 20.04 及以上）
* 编译器：

  * GCC ≥ 9.0（Linux）
  * MSVC 2019 及以上（Windows）
* Qt SDK：Qt 5.15.x（含 Qt Widgets 模块）
* 构建工具：

  * CMake ≥ 3.15（或使用 qmake）

---

## 3. 安装依赖

### 3.1 Qt 安装

1. 前往 Qt 官方安装器下载 Qt Online Installer
2. 选择 Qt 5.15.x 版本
3. 勾选以下组件：

   * Qt Widgets
   * 对应平台编译套件（MinGW / MSVC / GCC）
4. 完成安装并配置环境变量

### 3.2 编译器检查

#### Windows

* 确保 MSVC 或 MinGW 已正确安装
* 在命令行中执行：

```bash
cl
```

或

```bash
g++ --version
```

#### Linux

```bash
g++ --version
cmake --version
```

## 4. 项目结构说明

```text
Bookstore/
├── doc/                # 文档文件夹（需求/设计/说明等）
├── include/            # 头文件
├── src/                # C++ 源码
├── frontend/           # 前端源码
├── CMakeLists.txt      # CMake 构建脚本
├── README.md           # 项目说明
└── .gitignore          # Git 忽略列表

```

---

## 5. 编译与安装

### 5.1 使用 Qt Creator（推荐）

1. 打开 Qt Creator
2. 选择“打开项目”，加载 CMakeLists.txt
3. 配置构建套件（Kit）
4. 点击“构建”并运行

### 5.2 命令行方式（CMake）

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

生成的可执行文件位于 `build/` 目录下。

---

## 6. 初始化与首次运行

1. 运行 `Bookstore` 可执行程序
2. 系统将自动初始化数据文件
3. 若为首次运行，将生成默认管理员账户

---

## 7. 卸载说明

* 删除程序目录
* 清除生成的数据文件（保存在build/data目录）
